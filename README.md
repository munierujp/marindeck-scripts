# marindeck-scripts

MarinDeckのカスタムJavaScript用スクリプト

## 使い方

1. 各スクリプトの中身をコピー
2. MarinDeckのカスタムJavaScriptの入力欄にペースト
3. MarinDeckの画面をリロード
